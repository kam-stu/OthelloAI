# OthelloAI

This project is a fullstack website using:
* Flask 
* React 

This website implements an AI playing Othello based on minimax with a custom heuristic

AI for this program was used for the following:
1) Optimizing heuristic values 
2) Generating CSS for the styling of this website
3) Re-learning React (for the 2nd or 3rd time)
4) Navigating JavaScript quirks

Other resources (including GeeksForGeeks) was used for the following:
1) Learning more about JavaScript and how to use it
2) Learning more about React
3) Getting psuedo code for minimax algorithm
4) CSS styling
